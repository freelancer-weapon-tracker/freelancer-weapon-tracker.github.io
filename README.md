# Hitman Freelancer Weapon Tracker
A simple tool to help you find and track weapons and items in the Freelancer game mode of HITMAN World of Assassination.

Note: The interface is designed to work best on phone screens.

Use it here: [Freelancer Weapon Tracker](https://freelancer-weapon-tracker.github.io/)
